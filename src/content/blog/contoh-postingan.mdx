# Contoh Postingan Blog Baru

Ini adalah paragraf pertama dari postingan blog Anda. Anda bisa menulis konten menggunakan Markdown.

## Menggunakan Markdown

Anda bisa menggunakan semua fitur Markdown seperti:

- List item 1
- List item 2
- List item 3

### Kode

```javascript
// Anda juga bisa menambahkan blok kode
const hello = "world";
console.log(hello);
```

### Gambar

Anda bisa menambahkan gambar dengan format Markdown:

![Contoh gambar](/images/example.jpg)

Atau menggunakan komponen Image dari Astro:

import { Image } from 'astro:assets';

<Image src="/images/example.jpg" alt="Contoh gambar dengan komponen Astro" />
